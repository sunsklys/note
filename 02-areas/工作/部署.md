## 日志

### 清理

```bash
# 1. 先看（不带 -delete）
find . -maxdepth 1 -type f -name '*.log' ! -newermt '2026-06-12' -ls

# 2. 看完确认是死日志，再删
find . -maxdepth 1 -type f -name '*.log' ! -newermt '2026-06-12' -delete
```

| 项目       | 路径                    |
| ---------- | ----------------------- |
| 应用日志   | /var/data/logs/saas_dev |
| Nginx 配置 | /etc/nginx/conf.d       |
| 机器日志   | /var/www/dist/logs      |

## 视频处理

转换 dav → mp4：

```bash
ffmpeg -i input_file_name.dav -c:v libx264 -crf 20 -c:a aac -strict experimental -b:a 192k output_file_name.mp4
```

截取片段：

```bash
ffmpeg -i output_video.mp4 -ss 00:05:00 -t 00:01:20 output-2.mp4
```
